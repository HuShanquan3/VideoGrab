# Chrome扩展：视频抓取器
![alt](preview.jpg)  
解析视频网页，抓取视频地址，在新标签页中播放(mp4)或者在媒体播放器中播放(flv,m3u8...)，减少Flash播放视频CPU占用高的问题，让风扇安静下来，还可以无视广告。  
致谢：视频提供商。  
参考：[YouKuDownLoader](https://github.com/zhangn1985/ykdl)。
